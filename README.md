# Lovable Base44 Manus 1.6 - True Multi-Agent Architecture

Full implementation of Lovable-like system with FastAPI backend, multi-agent LangGraph, self-repair, Docker sandbox, etc.

## 🚀 Quick Start

### Backend
```bash
cd backend
pip install -r requirements.txt
python -m uvicorn app.main:app --reload --port 8000
```

### Streamlit UI
```bash
streamlit run streamlit_app.py
```

Open http://localhost:8501
